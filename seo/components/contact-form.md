# فرم تماس — Lead Capture

## فیلدها

| فیلد | نوع | الزامی |
|------|-----|--------|
| نام و نام خانوادگی | text | خیر |
| شماره موبایل | tel | **بله** |
| نوع درخواست | select | خیر |
| نوع کسب‌وکار | select | خیر |
| آدرس سایت فعلی | url | خیر |
| توضیحات | textarea | خیر |

## گزینه‌های «نوع درخواست»

- مشاوره رایگان
- ساخت فروشگاه جدید
- طراحی فروشگاه
- مهاجرت از پلتفرم دیگر
- پشتیبانی

## گزینه‌های «نوع کسب‌وکار»

- پوشاک و مد
- لوازم آرایشی
- لوازم خانگی
- مواد غذایی
- فروشنده اینستاگرام
- سایر

## پیام موفقیت

```
درخواست شما ثبت شد. حداکثر تا ۲۴ ساعت کاری با شما تماس می‌گیریم.
```

## اعلان به ادمین (ایمیل/تلگرام)

```
درخواست جدید فروشگاه:
نام: {name}
موبایل: {phone}
نوع: {request_type}
کسب‌وکار: {business_type}
سایت: {website}
توضیحات: {message}
```

## اعتبارسنجی

- موبایل: ۱۱ رقم، شروع با 09
- حداقل طول توضیحات: ۱۰ کاراکتر (اختیاری)
- Rate limit: ۳ درخواست در ساعت per IP
