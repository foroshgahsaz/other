# المان‌های اعتماد (Trust Signals)

## برای صفحه اصلی و لندینگ

```
✅ بیش از [X] فروشگاه پیاده‌سازی شده
✅ پشتیبانی فارسی
✅ تحویل ۱ تا ۳ هفته‌ای
✅ درگاه پرداخت ایرانی
✅ آموزش پنل مدیریت
✅ مشاوره اولیه رایگان
```

## بخش نظرات مشتری (نمونه — جایگزین با نظر واقعی)

### نظر ۱
> «فروشگاه پوشاکم را در ۲ هفته تحویل گرفتم. پنل خیلی ساده است.»
> — سارا م.، فروشگاه لباس زنانه

### نظر ۲
> «از اینستاگرام به فروشگاه独立 شدم. فروشم ۳ برابر شد.»
> — علی ر.، لوازم خانگی

### نظر ۳
> «پشتیبانی عالی. هر سوالی داشتم جواب دادند.»
> — مریم ک.، آرایشی

## Schema Review (برای نظرات واقعی)

```json
{
  "@type": "Review",
  "author": {"@type": "Person", "name": "سارا م."},
  "reviewRating": {"@type": "Rating", "ratingValue": "5"},
  "reviewBody": "فروشگاه پوشاکم را در ۲ هفته تحویل گرفتم."
}
```

## لوگوهای اعتماد

- اینماد (پس از دریافت)
- درگاه‌های پرداخت: زرین‌پال، آیدی‌پی
- «پشتیبانی ۲۴ ساعته» (در صورت واقعی)
