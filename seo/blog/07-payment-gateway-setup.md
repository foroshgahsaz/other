# آموزش اتصال درگاه پرداخت به فروشگاه اینترنتی

بدون **درگاه پرداخت**، فروشگاه اینترنتی فقط یک کاتالوگ است. در این راهنما مراحل اتصال درگاه‌های معتبر ایرانی را توضیح می‌دهیم.

> اگر می‌خواهید ما درگاه را برایتان وصل کنیم: [سفارش فروشگاه](/contact)

---

## پیش‌نیازها

قبل از درخواست درگاه:

| مورد | توضیح |
|------|--------|
| اینماد | نماد اعتماد الکترونیکی |
| دامنه | با SSL فعال |
| صفحات قانونی | حریم خصوصی، قوانین، تماس |
| حساب بانکی | مطابق نوع درگاه |

---

## درگاه‌های پرکاربرد در ایران

### ۱. زرین‌پال (Zarinpal)
- پرکاربردترین
- ثبت‌نام آنلاین
- کارمزد حدود ۱٪

### ۲. آیدی‌پی (IDPay)
- رابط ساده
- مناسب استارتاپ

### ۳. سامان (SEP)
- بانکی
- مناسب شرکت‌ها

### ۴. ملت (Behpardakht)
- گسترده در فروشگاه‌های بزرگ

---

## مراحل اتصال (عمومی)

```
۱. ثبت‌نام در سایت درگاه
۲. ارسال مدارک + آدرس فروشگاه
۳. تایید و دریافت Merchant ID / API Key
۴. وارد کردن کلید در پنل فروشگاه
۵. تست تراکنش (مبلغ کم)
۶. فعال‌سازی نهایی
```

---

## اتصال به فروشگاه‌ساز

وقتی فروشگاه را با [فروشگاه‌ساز](/store-builder) سفارش می‌دهید:

✅ درگاه در قیمت پایه گنجانده شده  
✅ تست تراکنش انجام می‌شود  
✅ آموزش مشاهده گزارش پرداخت  

---

## نکات امنیتی

- **هرگز** API Key را عمومی نکنید
- فقط از درگاه‌های معتبر استفاده کنید
- SSL الزامی است
- گزارش تراکنش‌ها را روزانه چک کنید

---

## مشکلات رایج

| مشکل | راه‌حل |
|------|--------|
| رد اینماد | تکمیل صفحات قانونی |
| خطای callback | بررسی URL در تنظیمات درگاه |
| تراکنش ناموفق | تست با مبلغ ۱۰۰۰ تومان |

---

## سوالات متداول

**بدون اینماد درگاه می‌شود؟**  
برای درگاه‌های رسمی معمولاً خیر.

**چند درگاه همزمان؟**  
بله. مشتری انتخاب می‌کند.

**کارمزد چقدر است؟**  
۱ – ۲٪ هر تراکنش (بسته به درگاه).
